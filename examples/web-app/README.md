# ViePilot Example: Web App

A Next.js todo application built with ViePilot autonomous mode.

**Setup time**: ~3 minutes  
**Stack**: Next.js 14, TypeScript, Tailwind CSS, SQLite

## What This Demonstrates

- `/vp-brainstorm` → gathering web app requirements
- `/vp-crystallize` → generating project phases
- `/vp-auto` → autonomous Next.js scaffolding
- `/vp-status` → progress tracking through frontend phases

## Quick Start

```bash
# 1. Clone ViePilot and install
git clone https://github.com/0-CODE/viepilot && cd viepilot && npm install && node bin/viepilot.cjs install --target cursor-agent --yes

# 2. Create a new project directory
mkdir my-todo-app && cd my-todo-app && git init

# 3. Start ViePilot brainstorm
# In Cursor: /vp-brainstorm Build a todo web app with Next.js

# 4. Crystallize into phases
# In Cursor: /vp-crystallize

# 5. Run autonomous execution
# In Cursor: /vp-auto
```

## Key ViePilot Commands Used

```bash
# Check progress anytime
/vp-status

# If something breaks
/vp-debug investigate: component not rendering

# Roll back to last checkpoint
/vp-rollback

# Add a feature mid-project
/vp-request --feature Add dark mode toggle
```
