# Task 74.3 — scripts/gen-proposal-pptx.cjs: palette-driven + rich layouts + 3 PPTX variants

## Goal
Extend `gen-proposal-pptx.cjs` to support 3 colour palettes and 3 rich slide layouts (timeline-gantt, team-card, investment-visual). Generate 3 PPTX variants for `project-proposal` type.

## Paths
```
scripts/gen-proposal-pptx.cjs
```

## Implementation Notes

### 1. Palette map (add near top, after existing `C` const)
```js
const PALETTES = {
  'modern-tech': { bg: '1a1f36', accent: '4f6ef7', accentDim: '2a3f9e', muted: '8892b0', mutedDim: '5a637a' },
  'enterprise':  { bg: '1a1f36', accent: 'c9a84c', accentDim: '9b7b2a', muted: '9aa0b0', mutedDim: '6b7080' },
  'creative':    { bg: '0d1020', accent: 'f7376e', accentDim: 'b02050', muted: '9b8fb0', mutedDim: '6b5f80' },
};
```

### 2. makePres(palette) — accept palette param
- Refactor `makePres()` to accept an optional `palette` key string (default: `'modern-tech'`).
- Use `PALETTES[palette]` for all colour references in slide master.

### 3. Rich layout: addTimelineGanttSlide(pres, { heading, phases, C })
A gantt-inspired timeline slide for project-proposal:
- Heading bar at top
- Each phase: label + filled rect bar (proportional width) + "N wks" note to the right
- Phases array: `[{ label, duration, note }]` (up to 5 items)
- Use palette accent colour for bars

### 4. Rich layout: addTeamCardSlide(pres, { heading, members, C })
Team card grid slide:
- Heading bar
- Up to 4 member cards in 2×2 grid (each: role bold, name, 1-line background)
- Cards: rounded-rect background (charcoal), accent left border stripe

### 5. Rich layout: addInvestmentVisualSlide(pres, { heading, total, breakdown, C })
Investment visual slide:
- Large total value centred, accent colour, large font (48pt)
- 3-column breakdown below: value + label + note (same as existing `data` layout)
- Accent bar separator between total and breakdown

### 6. Generate 3 project-proposal variants
At the bottom, in `main()`, for `project-proposal` type generate 3 files:
- `project-proposal-modern-tech.pptx` — using `modern-tech` palette + timeline-gantt + team-card + investment-visual
- `project-proposal-enterprise.pptx` — using `enterprise` palette + same rich layouts
- `project-proposal-creative.pptx` — using `creative` palette + same rich layouts

Keep existing single-palette generation for the other 3 types (`tech-architecture`, `product-pitch`, `general`) unchanged — they continue to generate `{type}.pptx`.
