# Task 22.1 — Define deterministic project slug for tag prefix

## Objective
Create a stable, normalized project slug that can prefix checkpoint tags without collisions across repositories.

## Paths
- `lib/cli-shared.cjs`
- `bin/vp-tools.cjs`

## File-Level Plan
- Add helper to derive slug from git repo basename (fallback to project root folder name).
- Normalize slug to lowercase alphanumeric + dash to keep git tag safe.
- Expose helper for reuse in CLI commands and workflow docs.

## Verification
```bash
node -e "const s=require('./lib/cli-shared.cjs'); console.log(s.getProjectSlug())"
node bin/vp-tools.cjs tag-prefix --raw
```
