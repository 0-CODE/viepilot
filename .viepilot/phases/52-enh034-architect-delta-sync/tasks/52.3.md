# Task 52.3 — templates/architect/architect-actions.js: markStale() + stale injection

**Phase**: 52 | **Status**: planned

## Paths
- `templates/architect/architect-actions.js`

## Objective
Extend architect-actions.js with:
1. `markStale(id, reason)` — marks a specific item as stale/needing-sync (sets
   data-arch-stale="true", injects the gap badge)
2. Stale injection on DOMContentLoaded — scan for `[data-arch-stale="true"]` already
   present in HTML (set by the sync step) and inject the gap badges automatically

This enables the workflow to write `data-arch-stale="true"` attributes into the HTML
and have the visual badge appear without manual JS calls.

## File-Level Plan

### architect-actions.js — extend IIFE

**Add after the existing `inject()` function:**

```js
// ENH-034: Stale/gap badge injection

function injectStaleBadges() {
  document.querySelectorAll('[data-arch-stale="true"]').forEach(el => {
    const reason = el.getAttribute('data-arch-stale-note') || 'gap detected';
    const badge = document.createElement('span');
    badge.className = 'arch-gap-badge';
    badge.textContent = '⚠ gap';
    badge.title = reason;

    if (el.tagName === 'TR') {
      const firstTd = el.querySelector('td');
      if (firstTd && !firstTd.querySelector('.arch-gap-badge')) {
        firstTd.appendChild(badge);
      }
    } else {
      const h2 = el.querySelector('h2');
      if (h2 && !h2.querySelector('.arch-gap-badge')) {
        h2.appendChild(badge);
      }
    }
  });
}

function markStale(id, reason) {
  const el = document.querySelector('[data-arch-id="' + id + '"]');
  if (!el) return;
  el.setAttribute('data-arch-stale', 'true');
  if (reason) el.setAttribute('data-arch-stale-note', reason);
  injectStaleBadges();
}
```

**Update DOMContentLoaded handler** to also call `injectStaleBadges()` after `inject()`:
```js
document.addEventListener('DOMContentLoaded', function () {
  inject();
  injectStaleBadges();
});
```

**Expose markStale globally** (for manual console use):
```js
window.vpMarkStale = markStale;
```
