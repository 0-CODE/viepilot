# Task 52.2 — templates/architect/style.css: .arch-stale + .arch-gap-badge classes

**Phase**: 52 | **Status**: planned

## Paths
- `templates/architect/style.css`

## Objective
Add visual CSS classes for items that have been flagged as stale/gap-detected but not yet
synced. These are distinct from the existing `.updated` class (green = recently changed).
Stale/gap = amber/orange = "brainstorm found something here but HTML not updated yet."

## File-Level Plan

### style.css — add stale gap indicator classes

```css
/* ENH-034: Architect gap/stale indicators */
.arch-gap-badge {
  display: inline-block;
  font-size: 10px;
  font-weight: 600;
  padding: 1px 5px;

  background: rgba(245, 158, 11, 0.15);   /* amber */
  color: #f59e0b;

}

[data-arch-stale="true"],
tr[data-arch-stale="true"] td:first-child,

  border-left: 3px solid #f59e0b !important;
}

[data-arch-stale="true"] .arch-gap-badge,

  display: inline-block;
}

/* Light mode overrides */
html.light .arch-gap-badge {

  border-color: rgba(217, 119, 6, 0.3);
```

Key design decisions:
- Amber (#f59e0b) — distinct from `.updated` (yellow left border / green badge)
  and `.arch-btn-approve` (green) / `.arch-btn-edit` (blue)
- Badge text "⚠ gap" (short, visible at a glance)
- Left border reinforcement for table rows and cards
