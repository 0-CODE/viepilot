## Detection Algorithm
```
scope_drift_check(task):
  modified_files = git_diff_names()
  declared_scope = task.write_scope
  
  violations = []
  for file in modified_files:
    if not any(file.startswith(scope) for scope in declared_scope):
      violations.append(file)
  
  if violations:
    log scope_drift event to HANDOFF.log
    control_point(reason=f"Scope drift: {violations} outside {declared_scope}")
  else:
    return CLEAN
```
