## State Definitions
```
not_started:   entry=none; transitions=[on_start → executing]
executing:     entry=run_sub_tasks; transitions=[on_pass → pass | on_fail → recovering_l1]
recovering_l1: entry=lint_fix; transitions=[on_pass → pass | on_exhausted → recovering_l2]
recovering_l2: entry=test_fix; transitions=[on_pass → pass | on_exhausted → recovering_l3]
recovering_l3: entry=scope_reduce; transitions=[on_pass → pass | on_exhausted → control_point | L3.block → control_point]
control_point: entry=surface_to_user; transitions=[on_user_resolve → executing | on_user_skip → skip]
pass:          entry=update_state; transitions=[on_next_task → executing | on_phase_done → phase_complete]
skip:          entry=document_reason; transitions=[on_next_task → executing | on_phase_done → phase_complete]
```
