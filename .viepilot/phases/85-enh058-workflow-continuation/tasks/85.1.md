# Task 85.1 — AUQ continuation in `evolve.md` Step 5 + `request.md` Step 6 + SKILL.md AUQ tables

## Objective
Replace the static "Next:" text blocks at the end of `evolve.md` and `request.md` with
`AskUserQuestion` continuation prompts that actively offer the next action and can
immediately invoke the chosen skill.

## Paths
- `viepilot/workflows/evolve.md`
- `viepilot/workflows/request.md`
- `skills/vp-evolve/SKILL.md`
- `skills/vp-request/SKILL.md`

## Changes

### 1. `viepilot/workflows/evolve.md` — Step 5

**Find** the current "Confirm & Suggest Next" step which ends with something like:
```markdown
Next action: /vp-auto --from {new_phase}
```

**Replace** with:
```markdown
### Step 5: Confirm & Suggest Next

Output the evolution summary (mode, phase number, version bump, files changed).

Then — **Claude Code adapter REQUIRED**:

**Invoke AskUserQuestion:**
```
question: "Phase {N} planned ({REQUEST_ID} → {VERSION}). What would you like to do next?"
options:
  - label: "Execute now → /vp-auto"
    description: "Start implementing Phase {N} immediately in this session (Recommended)"
  - label: "Create another request → /vp-request"
    description: "Log more requests before implementing"
  - label: "Done for now"
    description: "Exit — run /vp-auto later to start implementation"
```

**On selection:**
- "Execute now → /vp-auto": invoke vp-auto skill
- "Create another request → /vp-request": invoke vp-request skill
- "Done for now": print brief summary and exit

**Text fallback (non-Claude Code):**
```
