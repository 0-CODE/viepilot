## Gap A — Auto-populate Logic
```
type inference:
  description contains "create/add/implement/build" → type: build
  description contains "fix/bug/repair" → type: fix
  description contains "refactor/cleanup" → type: refactor
  description contains "test/verify" → type: test
  description contains "doc/documentation/readme" → type: docs
  default → type: build

write_scope inference:
  from ROADMAP.md task "Paths" or description keywords
  e.g., "refactor TRACKER.md" → write_scope: ["templates/project/TRACKER.md"]
  e.g., "update autonomous.md" → write_scope: ["workflows/autonomous.md"]

recovery_budget:
  = task complexity from ROADMAP.md table
```

## Gap G — Compliance Domain Detection
```
compliance_domains:
  auth:     paths containing "/auth/", "/authorization/", "jwt", "oauth", "session", "rbac"
  payment:  paths containing "/payment/", "/billing/", "stripe", "invoice", "subscription"
  data:     paths containing "/migrations/", "schema", "foreign_key"
  crypto:   paths containing "/crypto/", "encrypt", "hash", "password", "bcrypt", "tls"

For each task in TASK.md:
  if any write_scope path matches compliance_domain:
    set recovery_overrides.L3.block = true
    set recovery_overrides.L3.reason = "{domain} domain — scope reduction unsafe"
```
