# Task 81.6 — AUQ Enforcement in 4 Workflows + 4 SKILL.md (ENH-055)

## Objective
Change all `AskUserQuestion` prompt blocks in the 4 affected workflows from
"use AUQ if available" to "Claude Code REQUIRED — text is the non-Claude-Code fallback only".
Update the corresponding SKILL.md docs to reflect enforcement.

## Paths
- `workflows/evolve.md`
- `workflows/request.md`
- `workflows/brainstorm.md`
- `workflows/crystallize.md`
- `skills/vp-evolve/SKILL.md`
- `skills/vp-request/SKILL.md`
- `skills/vp-brainstorm/SKILL.md`
- `skills/vp-crystallize/SKILL.md`

## File-Level Plan

### Pattern: replace ALL AUQ prompt preamble blocks across all 4 workflows

**Current pattern (in every AUQ block):**
```
> **Adapter-aware prompt:**
> - **Claude Code (terminal):** use `AskUserQuestion` tool — spec:
>   - question: "..."
>   ...
> - **Cursor / Codex / Antigravity / other:** use text menu below
```

**Replace with this enforced pattern:**
```
> **Adapter-aware prompt — Claude Code (terminal): REQUIRED**
> Call `AskUserQuestion` tool. Only fall back to text menu if the tool returns an error or is unavailable.
> - question: "..."
> - (other AUQ spec fields unchanged)
>
> **Cursor / Codex CLI / Antigravity / other adapters:** AUQ not available — use text menu below.
```
