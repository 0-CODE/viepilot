# Task 83.4 — Wire doc-sync-agent into `autonomous.md` bulk-edit pattern

## Objective
Add a doc-sync-agent invocation pattern to `viepilot/workflows/autonomous.md` for
tasks that require identical changes across many files (e.g., adding adapter rows to
all 17 SKILL.md files, updating version numbers in banners). This prevents the
17×-repeated-edit pattern seen in Phase 82 (ENH-056) and Phase 84 (FEAT-019).

## Paths
- `viepilot/workflows/autonomous.md`

## Changes Required

### 1. Add doc-sync-agent pattern to bulk-edit task handling
In the task execution section of autonomous.md, add a recognition pattern:

```markdown
#### Bulk-Edit Task Detection
If a task description matches ANY of:
- "update all N {filetype} files"
- "add {row/section} to all skills"
- "sync {content} across {N} files"
- Task Paths list contains ≥5 identical file types

→ **Invoke: doc-sync-agent** instead of N sequential edits:
- File pattern: {glob from task Paths}
- Change: {description from task objective}
- Mode: `insert` | `replace` | `append`

doc-sync-agent returns: list of updated files + diff count.
Report result, do not re-edit individually.
```
