# Task 87.1 — Add AUQ preload block to 5 SKILL.md files

## Objective
Insert a `ToolSearch` preload instruction into the AUQ adapter section of each SKILL.md that uses AskUserQuestion, so the model always loads the tool schema before attempting the first interactive prompt.

## Paths
- `skills/vp-request/SKILL.md`
- `skills/vp-evolve/SKILL.md`
- `skills/vp-auto/SKILL.md`
- `skills/vp-brainstorm/SKILL.md`
- `skills/vp-crystallize/SKILL.md`

## Implementation Notes

### Standard preload block to insert
In each SKILL.md, locate the **AUQ adapter compatibility block** — specifically the line:

```
**Claude Code (terminal):** Always call `AskUserQuestion` first. Only fall back to the plain-text menu if the tool returns an error or is unavailable.
```

**Replace** that line with:

```
**Claude Code (terminal) — AUQ preload required:**
Before the first interactive prompt, call `ToolSearch` with `query: "select:AskUserQuestion"` to load the deferred tool schema. Only after the schema is loaded can `AskUserQuestion` be invoked. If `ToolSearch` itself returns an error, fall back to plain-text numbered list.
```

### Files and exact location
Each of the 5 files has this block immediately after the adapter compatibility table, before `**Prompts using AskUserQuestion in this skill:**`. The replacement is purely additive — it clarifies the preload requirement without removing existing guidance.
