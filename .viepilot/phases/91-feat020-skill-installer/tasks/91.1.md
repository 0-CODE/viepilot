### Channel detection

```js
function detectChannel(source) {
  if (source.startsWith('github:')) return 'github';
  if (source.startsWith('./') || source.startsWith('/') || source.startsWith('..')) return 'local';
  return 'npm';  // default: bare name or npm: prefix
}
```

### npm channel

```js
// Normalize: strip "npm:" prefix if present
const pkgName = source.replace(/^npm:/, '');
const skillId = pkgName.split('/').pop().replace(/^vp-skills-/, '');

// For each adapter skillsDir:
//   1. Create target dir: path.join(skillsDir, skillId)
//   2. Copy SKILL.md from npm package (use `npm pack` + extract, or npm install + read node_modules)
//   Strategy: use execSync(`npm show ${pkgName} dist-tags.latest`) to verify existence
//             then execSync(`npm pack ${pkgName} --dry-run`) — or simpler:
//             install into temp dir → copy SKILL.md out
```

For MVP simplicity: use `npm pack` into a temp dir, extract SKILL.md and any skill assets.

### Post-install

After any successful install:
```js

scanSkills(home);  // refresh registry
```

### Source recording

After install, write source to skill dir as `skill-meta.json`:
```json

```

Used by `updateSkill()` to re-install from same source.
