## Objective
Expose the three skill management commands as CLI subcommands.

## Paths

## Implementation Notes

### Commands to add

### printHelp() additions
```
  install-skill <source>           Install a third-party skill (npm:pkg, github:org/repo, ./path)
  uninstall-skill <id>             Remove a skill from all adapter dirs
  update-skill <id>                Re-install skill from its original source
```

### CLI wiring pattern (follow existing scan-skills pattern)
```js
if (command === 'install-skill') {
  const source = rest[0];
  if (!source) { console.error('Usage: vp-tools install-skill <source>'); process.exit(1); }
  const { installSkill } = require('../lib/skill-installer.cjs');
  const result = installSkill(source);
  // print summary
  process.exit(result.ok ? 0 : 1);
}
```
