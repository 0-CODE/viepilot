**Guard rule (BUG-018):**
- Architect Mode auto-activate heuristic (≥3 components / ≥1 stack mention) **MUST NOT fire independently** after this selection is made.
- If heuristic would have fired before scope lock: defer until Step 2B runs.
- If user selects "Neither" or "UI Direction only": suppress Architect auto-activate for this session.
```
