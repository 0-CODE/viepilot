# Task 38.1 — buildInstallPlan: claudeViepilotDir + mkdir/copy steps

## Objective
Trong `buildInstallPlan()`, khi `installClaudeSkills` là true:
1. Tạo `claudeViepilotDir = path.join(home, '.claude', 'viepilot')`
2. Thêm mkdir steps cho tất cả subdirs của `~/.claude/viepilot/`
3. Thêm copy steps mirror `workflows/`, `templates/project/`, `templates/phase/`, `bin/` (vp-tools.cjs + viepilot.cjs), `lib/cli-shared.cjs` từ packageRoot sang `~/.claude/viepilot/`
4. Expose `claudeViepilotDir` trong `paths` return object
5. Update `formatPlanLines` để hiển thị `viepilot (Claude Code): {claudeViepilotDir}` khi applicable

## Paths
- `lib/viepilot-install.cjs`

## File-Level Plan

### 1. Sau dòng 96 (`const claudeSkillsDir = ...`):
```js
const claudeViepilotDir = installClaudeSkills
  ? path.join(home, '.claude', 'viepilot')
  : null;
```

### 2. Trong block `if (installClaudeSkills && claudeSkillsDir)` (sau dòng 155, sau các skill copy steps), thêm:
```js
if (claudeViepilotDir) {
  // mkdir subdirs
  for (const sub of ['workflows', path.join('templates','project'), path.join('templates','phase'), 'bin', 'lib']) {
    steps.push({ kind: 'mkdir', path: path.join(claudeViepilotDir, sub) });
  }

  // mirror workflows
  for (const ent of listDirEntries(root, 'workflows')) {
    const src = path.join(root, 'workflows', ent.name);
    const dest = path.join(claudeViepilotDir, 'workflows', ent.name);
    steps.push(ent.isDirectory()
      ? { kind: 'copy_dir', from: src, to: dest }
      : { kind: 'copy_file', from: src, to: dest });
  }

  // mirror templates/project
  for (const ent of listDirEntries(root, path.join('templates', 'project'))) {
    const src = path.join(root, 'templates', 'project', ent.name);
    const dest = path.join(claudeViepilotDir, 'templates', 'project', ent.name);
    steps.push(ent.isDirectory()
      ? { kind: 'copy_dir', from: src, to: dest }
      : { kind: 'copy_file', from: src, to: dest });
  }

  // mirror templates/phase
  for (const ent of listDirEntries(root, path.join('templates', 'phase'))) {
    const src = path.join(root, 'templates', 'phase', ent.name);
    const dest = path.join(claudeViepilotDir, 'templates', 'phase', ent.name);
    steps.push(ent.isDirectory()
      ? { kind: 'copy_dir', from: src, to: dest }
      : { kind: 'copy_file', from: src, to: dest });
  }

  // mirror bin + lib
  for (const f of ['vp-tools.cjs', 'viepilot.cjs']) {
    steps.push({ kind: 'copy_file', from: path.join(root, 'bin', f), to: path.join(claudeViepilotDir, 'bin', f) });
  }
  steps.push({
    kind: 'copy_file',
    from: path.join(root, 'lib', 'cli-shared.cjs'),
    to: path.join(claudeViepilotDir, 'lib', 'cli-shared.cjs'),
  });
}
```
