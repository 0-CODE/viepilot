# Task 54.1 — lib/hooks/brainstorm-staleness.cjs

**Phase**: 54 | **Status**: planned

## Paths
- `lib/hooks/brainstorm-staleness.cjs`

## Objective
Create the ViePilot staleness hook for Claude Code's `Stop` event. When Claude Code finishes
each AI response turn, this script fires automatically. It reads session context from stdin
JSON, locates the active brainstorm session files in `cwd`, scans architect HTML templates
for items that may have become stale relative to recent session content, and marks them with
`data-arch-stale="true"` by patching the HTML on disk.

Option A (flag-only) — does NOT auto-rewrite content; just marks items as stale. Non-blocking: exit 0 always.

**Input reading:**
```js
process.stdin.setEncoding('utf8');
let raw = '';
process.stdin.on('data', (chunk) => { raw += chunk; });
process.stdin.on('end', () => {
  let hookData = {};
  try { hookData = JSON.parse(raw); } catch (_e) { /* no stdin = dev/test run */ }
  run(hookData).catch((e) => {
    process.stderr.write(`[viepilot-hook] error: ${e.message}\n`);
  }).finally(() => process.exit(0));
});
process.stdin.on('error', () => process.exit(0));
```

**Main `run(hookData)` function:**
```js
async function run(hookData) {
  const cwd = hookData.cwd || process.cwd();
  const session = await findActiveSession(cwd);
  if (!session) return; // no active session — nothing to do

  const architectDir = path.join(cwd, 'templates', 'architect');
  // Also check installed location
  const installDir = path.join(require('os').homedir(), '.claude', 'viepilot', 'templates', 'architect');
  const targetDir = fs.existsSync(architectDir) ? architectDir :
                    fs.existsSync(installDir)   ? installDir : null;
  if (!targetDir) return;

  const stalePages = detectStaleItems(session.sessionContent, targetDir);
  let patchCount = 0;
  for (const { page, reason } of stalePages) {
    const filePath = path.join(targetDir, page);
    if (fs.existsSync(filePath)) {
      const changed = markStaleInFile(filePath, reason);
      if (changed) patchCount++;
    }
  }

  if (patchCount > 0) {
    process.stderr.write(`[viepilot-hook] ⚠ Marked ${patchCount} architect page(s) stale\n`);
  }
}
```
