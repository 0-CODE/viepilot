## Canonical Pattern

Every question block in any ViePilot workflow must follow this structure:

```markdown
> **Adapter-aware prompt:**
> | Adapter | Method |
> |---------|--------|
> | Claude Code (terminal) | Use `AskUserQuestion` tool (spec below) |
> | Cursor / Codex / Antigravity / other | Use text list below |

**AskUserQuestion spec** (Claude Code only):
- question: "{exact question text}"
- header: "{≤12 char label}"
- options: [{label, description}, ...] (2–4 options)
- multiSelect: true/false

**Text fallback** (all adapters):
1. Option A — {description}
2. Option B — {description}
...
```
