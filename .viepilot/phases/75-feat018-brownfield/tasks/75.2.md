### 11. Safety Rules Block

Add a `## Safety Rules` block inside the brownfield scanner step:

```
Scanner MUST skip the following (never read / glob into):
- .env  (live secrets)
- *.key, *.pem, *.p12, *.jks, id_rsa, id_ed25519
- node_modules/
- .git/
- target/
- build/
- dist/
- __pycache__/
- .venv/
- vendor/

Scanner MUST NOT write any files until user has confirmed the Scan Report.
Re-running --brownfield on existing .viepilot/ requires explicit user confirmation.
```
