# Task 13.3 - Introduce reusable detailed task template

## Objective
Add or update implementation task template so all coding tasks share consistent structure and quality controls.

## Template Sections
- Paths
- Implementation Notes
- Best Practices to Apply
- Do/Don't
- Verification (commands + expected output)
- State Update Checklist

## Inputs
- `templates/phase/TASK.md` (or equivalent)
- `workflows/autonomous.md`

## Acceptance Criteria
- [ ] Template exists with all mandatory sections.
- [ ] Autonomous workflow references template for implementation tasks.
