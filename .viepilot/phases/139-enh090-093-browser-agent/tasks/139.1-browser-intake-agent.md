## Objective

## File-Level Plan

### agents/claude-code/browser-intake-agent.md
YAML frontmatter:
```yaml
---
name: browser-intake-agent
description: >
  Reads publicly shared web URLs (Google Sheets, GitHub Issues, Jira, Trello, Notion)
  and extracts structured ticket/task rows for vp-intake. Uses vercel-labs/agent-browser
  for JS-rendered SPA content. Prerequisite: npx skills add vercel-labs/agent-browser.
model: claude-sonnet-4-6
tools:
  - Bash
  - Read
disallowedTools:
  - Edit
  - Write
  - WebSearch
  - WebFetch
  - Agent
---
```

System prompt: Accept `op` (read_url / screenshot / detect_type), `url`, `channel_config`.
- `detect_type`: pattern-match URL → google-sheets / github-issues / jira / trello / notion / generic-table
- `read_url`: open URL via `agent-browser open <url>`, `snapshot -i`, extract rows as JSON array `[{title, description, labels, priority, status}]`
- `screenshot`: capture page screenshot, save to `.viepilot/intake/screenshots/{timestamp}.png`
- If agent-browser not installed → emit JSON `{error: "agent-browser not installed. Run: npx skills add vercel-labs/agent-browser"}`
