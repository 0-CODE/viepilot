## Implementation Notes
1. **Profile slug**: `kebab-case`, chỉ `[a-z0-9-]`, max 64 chars; trùng với basename (không `.md`).
2. **Frontmatter keys** (profile file): `profile_id`, `display_name`, `org_tag`, `tags`, `last_updated` (ISO 8601 date), optional `locale`, `website`.
3. **profile-map.md**: markdown table cố định 6 cột + quy tắc append / cập nhật `last_used`.
4. **Binding**: `.viepilot/META.md` là source of truth; `HANDOFF.json` có thể mirror `viepilot_profile_id` sau 31.4 — trong 31.1 chỉ document priority (META first).
5. **Tách biệt** `templates/project/PROJECT-META.md` (artifact dài cho product) — không gộp; ViePilot dùng `VIEPILOT-META.md` nhỏ gọn.
