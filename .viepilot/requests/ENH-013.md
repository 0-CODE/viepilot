## Summary
Cập nhật lại `README.md` với metric dự án đúng trạng thái hiện tại, đồng thời chuyển `.viepilot` sang cơ chế local-only bằng cách cấu hình gitignore và gỡ tracking khỏi repository.

## Details

### Current behavior
- Metric trong `README.md` có dấu hiệu lệch so với codebase hiện tại theo vòng đời phase gần đây.
- Thư mục `.viepilot` vẫn đang được track trong git history/repository.

### Desired improvement
1. Chuẩn hoá lại metric trong `README.md` theo source of truth hiện tại (script/flow metric đang dùng trong dự án).
2. Bảo đảm quy trình cập nhật metric có thể chạy ổn định ở các lần cập nhật tiếp theo.
3. Thêm rule gitignore cho `.viepilot` để tránh tiếp tục commit artifact vận hành nội bộ.
4. Gỡ tracking `.viepilot` khỏi repository index nhưng giữ dữ liệu local.

## Acceptance Criteria
- [ ] `README.md` phản ánh đúng metric dự án tại thời điểm cập nhật.
- [ ] Có command/script hoặc verify steps rõ để re-sync metric lần sau.
- [ ] `.viepilot` được khai báo ignore trong cấu hình gitignore phù hợp.
- [ ] `.viepilot` không còn being tracked trên repository index (nhưng vẫn tồn tại local).

## Discussion
Yêu cầu ưu tiên dọn lại trạng thái "public repo hygiene": giữ tài liệu người dùng đúng số liệu và tách artifacts điều phối nội bộ khỏi nội dung cần publish.

## Resolution
Completed in M1.17 / Phase 21:
- Installed `cloc` and re-synced README metric via `npm run readme:sync`.
- Added `.viepilot/` to `.gitignore`.
- Ran `git rm -r --cached .viepilot` to untrack repository index while preserving local files.
