# BUG: ViePilot workflow/skills trả lời lẫn skill ngoài hệ sinh thái ViePilot

## Meta
- **ID**: BUG-004
- **Type**: Bug
- **Status**: done
- **Priority**: high
- **Created**: 2026-04-01
- **Reporter**: User
- **Assignee**: AI

## Summary
Khi dùng workflow của ViePilot, trợ lý có thể tham chiếu hoặc hướng dẫn skill ngoài hệ `vp-*`, gây nhiễu phạm vi framework và làm sai kỳ vọng “chỉ theo hệ sinh thái ViePilot”.

## Details
- **Actual behavior**:
  - Trong phiên làm việc, agent có thể liệt kê/gợi ý các skill không thuộc ViePilot (ví dụ từ global environment).
  - Một số phản hồi có thể trộn giữa skill framework ViePilot và skill bên ngoài.
- **Expected behavior**:
  - Trong ViePilot mode, chỉ được hướng dẫn và tham chiếu skill thuộc `vp-*` (nằm trong `skills/vp-*/SKILL.md` của package/repo hiện tại).
  - Skill ngoài hệ phải bị bỏ qua, hoặc chỉ hiển thị khi user bật explicit opt-in mode.
- **Repro (high-level)**:
  1. Chạy các lệnh skill như `/vp-auto`, `/vp-request`, `/vp-debug` trong môi trường có nhiều global skills.
  2. Quan sát phản hồi có đề cập skill không thuộc `vp-*`.
- **Impact**:
  - Mất tính nhất quán framework.
  - Tăng rủi ro route sai workflow.
  - Làm user khó tin cậy tài liệu/contract của ViePilot.

## Acceptance Criteria
- [x] Tất cả skill contracts `skills/vp-*/SKILL.md` có guard rõ: ưu tiên/giới hạn về `vp-*` khi ở ViePilot mode.
- [x] Các workflow chính (`autonomous`, `request`, `debug`, `documentation`, `audit`, `evolve`, `crystallize`, `brainstorm`) có rule lọc scope skill rõ ràng.
- [x] Có tài liệu mô tả behavior mặc định: chỉ `vp-*`; có cơ chế opt-in nếu muốn mở rộng.
- [x] Có test/verification rule để phát hiện phản hồi đề cập skill ngoài framework khi không được phép.

## Discussion
Yêu cầu: cập nhật đồng bộ toàn bộ skills để ràng buộc phạm vi chỉ dùng hệ sinh thái ViePilot, tránh drift do skill global của môi trường agent.

## Resolution
Done in M1.26 / Phase 30 (tasks 30.1 → 30.5): enforced vp-only default policy in skills and workflows, documented explicit opt-in behavior for external skills, and added regression tests (`tests/unit/vp-scope-policy-contracts.test.js`).
