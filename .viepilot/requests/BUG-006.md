## Summary
`buildInstallPlan()` chỉ copy `lib/cli-shared.cjs` cho cả 3 install targets (`claude-code`, `cursor-ide`, `cursor-agent`), bỏ qua 3 lib files còn lại. `vp-tools.cjs` crash khi require chúng.

## Details

### Actual behavior
```
node ~/.claude/viepilot/bin/vp-tools.cjs info  # claude-code
→ Error: Cannot find module '~/.claude/viepilot/lib/viepilot-info.cjs'

node ~/.cursor/viepilot/bin/vp-tools.cjs info  # cursor-ide / cursor-agent
→ Error: Cannot find module '~/.cursor/viepilot/lib/viepilot-info.cjs'
```

### Expected behavior
`vp-info`, `vp-update` và tất cả subcommands hoạt động bình thường trên mọi install target.

### Root cause
`buildInstallPlan()` có **2 blocks** đều chỉ copy `cli-shared.cjs`:
1. **Cursor block** (`viepilotDir` = `~/.cursor/viepilot/lib/`) — line ~258-262
2. **Claude Code block** (`claudeViepilotDir` = `~/.claude/viepilot/lib/`) — line ~190-194

Cả hai thiếu 3 copy steps:
- `lib/viepilot-info.cjs` — dùng bởi `vp-tools info`
- `lib/viepilot-update.cjs` — dùng bởi `vp-tools update`
- `lib/viepilot-install.cjs` — dùng bởi install subcommands
