## Summary
Nang cap `vp-audit` de kiem tra best practices theo tung tech stack va danh gia code quality theo guideline stack-specific. Khi chua co best-practice cache cho stack lien quan, `vp-audit` can tu dong research (web search tu nguon chinh thong/uy tin), tong hop ket qua, va de xuat cap nhat knowledge cache de dung lai.

## Current Feature
`vp-audit` da co 3-tier audit cho state consistency, documentation drift, va framework integrity. Tuy nhien, audit chua di sau vao compliance voi best practices theo tech stack va chua co fallback research khi thieu tri thuc stack.

## Current Behavior
- Kiem tra mismatch tai lieu/tracker/roadmap va framework artifacts.
- Khong danh gia codebase theo stack rules (Do/Don't, anti-patterns, conventions).
- Khong auto research neu stack moi/it duoc luu trong cache.

## Desired Improvement

### 1) Stack best-practice audit mode trong `vp-audit`
- Detect stack su dung trong du an/task scope (vd: Java + Spring + MyBatis, Node + Express, Python + FastAPI).
- Doi chieu code patterns voi stack guidelines (cau truc, layering, data access patterns, naming, test conventions).
- Bao cao theo muc do: critical/high/medium/low va mapping toi file/module lien quan.

### 2) Research fallback khi stack cache thieu/khong du
- Neu `~/.viepilot/stacks/{stack}/SUMMARY.md` khong ton tai hoac quality gate khong dat, chay `WebSearch` + `WebFetch`.
- Uu tien nguon official docs/specs va references uy tin.
- Tong hop output theo format chuan:
  - Do / Don't
  - Common pitfalls
  - Recommended structure
  - Implementation checklist
  - Code quality heuristics

### 3) Cache/update flow tu ket qua research
- De xuat (hoac auto-generate khi co flag) bo sung cache stack:
  - `~/.viepilot/stacks/{stack}/SUMMARY.md`
  - `~/.viepilot/stacks/{stack}/BEST-PRACTICES.md`
  - `~/.viepilot/stacks/{stack}/ANTI-PATTERNS.md`
  - `~/.viepilot/stacks/{stack}/SOURCES.md`
- Gan ngay cap nhat, do tin cay nguon, va chi dan refresh.

### 4) Alignment voi `vp-auto`
- Tham chieu preflight stack lookup da co trong `vp-auto` de audit va execution dung cung policy.
- `vp-audit` can co output de `vp-auto` co the consume lai (checklist/guardrails) truoc khi code.

## Value Proposition
- Tang do tin cay cua audit theo tung tech stack thay vi chi audit tai lieu/tracker.
- Giam nguy co anti-pattern va regressions code quality.
- Tao vong lap hoc hoi tu dong cho stack moi (research -> cache -> enforce).
- Tan dung toi uu token qua summary-first, detail-on-demand.
