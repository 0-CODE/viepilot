## Summary
Hiện tại không có slash command `/vp-skills` (agent skill) để quản lý global registry. Mọi thao tác scan/install/list đều phải qua `node bin/vp-tools.cjs` — agent không thể tự gọi được trong bất kỳ project nào mà không cần ở trong viepilot repo.

Cần tạo skill `skills/vp-skills/SKILL.md` để agent có thể dùng `/vp-skills` như một slash command độc lập trên bất kỳ adapter nào.

## Details

### Current behavior
- `vp-tools scan-skills` → chỉ chạy được khi ở trong viepilot repo với `node bin/vp-tools.cjs`
- `vp-tools list-skills` → tương tự
- Global registry `~/.viepilot/skill-registry.json` tồn tại nhưng không có skill-level interface
- Agent trong project khác (không phải viepilot repo) không có cách nào trigger scan hay đọc registry qua slash command

### Desired behavior
Tạo skill `skills/vp-skills/SKILL.md` để được install cùng framework:

```
/vp-skills scan          — Scan tất cả adapter dirs, refresh ~/.viepilot/skill-registry.json
/vp-skills list          — Hiển thị tất cả skills trong global registry
/vp-skills install <src> — Cài skill từ npm/github/local
/vp-skills uninstall <id>— Gỡ skill
/vp-skills update <id>   — Update skill từ source gốc
/vp-skills info <id>     — Xem chi tiết 1 skill (capabilities, best_practices, adapters)
```

Agent thực hiện bằng cách gọi:
```bash
node ~/.claude/viepilot/bin/vp-tools.cjs scan-skills
node ~/.claude/viepilot/bin/vp-tools.cjs list-skills
```
(dùng installed path, không cần ở trong viepilot repo)

### Why this is valuable
1. **Cross-project**: agent có thể quản lý skills từ bất kỳ project nào
2. **Agent-native**: slash command — cùng pattern với `/vp-auto`, `/vp-brainstorm`
3. **Fix BUG-016**: `vp-skills scan` + output JSON → workflow autonomous/brainstorm có thể dùng thay `Call loadRegistry()`
4. **Global registry**: một lần scan, tất cả projects dùng được

### Implementation sketch

`skills/vp-skills/SKILL.md`:
```markdown
# vp-skills

Quản lý global skill registry cho ViePilot.

## Commands
- scan    — refresh ~/.viepilot/skill-registry.json
- list    — show indexed skills
- install — npm/github/local
- uninstall
- update
- info

## Execution
Dùng installed vp-tools:
  node ~/.claude/viepilot/bin/vp-tools.cjs <command>
```
