## Summary
`/vp-proposal` hiện chỉ generate nội dung theo ngôn ngữ mặc định. User muốn có thể chọn ngôn ngữ trình bày của toàn bộ tài liệu proposal (AI-generated content: headings, bullets, speaker notes, docx sections). Ngôn ngữ đã dùng được lưu vào config để gợi ý lần sau.

## Details

### Behavior
- Flag: `--lang <code>` — ISO 639-1 (e.g. `vi`, `en`, `ja`, `fr`, `zh`)
- Nếu không truyền `--lang`: check `~/.viepilot/config.json` → `proposal.recentLangs` → gợi ý ngôn ngữ gần nhất
- Nếu config chưa có entry: default `en`
- Sau mỗi lần generate thành công: cập nhật `proposal.recentLangs` (max 5 entries, MRU order)

### Scope of language application
- **Default (option a)**: toàn bộ AI-generated content — slide headings, bullets, speaker notes, docx section titles, docx paragraph content, .md summary
- **Structure-only mode**: `--lang-content-only` — chỉ dịch nội dung; template placeholder labels (e.g. `{{Project Name}}`) và section structure headers giữ nguyên Tiếng Anh
- **Stock template labels** (e.g. slide master footer "ViePilot"): không dịch — luôn giữ nguyên

### Config integration
Tái sử dụng `lib/viepilot-config.cjs` (ENH-032 pattern):
```json
// ~/.viepilot/config.json
{
  "proposal": {
    "recentLangs": ["vi", "en"],   // MRU, max 5
    "defaultLang": "vi"             // most recently used
  }
}
```

### UX flow (no --lang provided)
```
Language for this proposal?
  1. Vietnamese (vi)  ← most recently used
  2. English (en)
  3. Other — enter code (e.g. ja, fr, zh, ko)
```

### Flag summary
| Flag | Description |
|------|-------------|
| `--lang <code>` | Language for all generated content. Default: from config or `en` |
| `--lang-content-only` | Translate content only; keep structure/placeholder labels in English |
