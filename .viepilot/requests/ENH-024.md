# Enhancement: ENH-024 — Đảm bảo ui-direction được đọc nhất quán qua crystallize → vp-auto

## Summary
Kết quả implementation (vp-auto) diverge khá xa với ui-direction đã brainstorm vì chuỗi truyền tải ui-direction bị đứt ở `crystallize` Step 10. Dù Step 1A đọc ui-direction đúng, thông tin không được "forward" vào execution context của task files.

## Details

### Gap 1 — crystallize Step 1A: trigger condition quá yếu
**File**: `~/.cursor/viepilot/workflows/crystallize.md` Step 1A  
**Hiện tại**: `"If brainstorm indicates frontend/UI scope, check for direction artifacts"`  
**Vấn đề**: "indicates" là heuristic — AI có thể miss signal và skip Step 1A, bỏ qua ui-direction hoàn toàn.  
**Fix**: Thêm điều kiện cứng: nếu `.viepilot/ui-direction/` tồn tại → Step 1A là **mandatory** (không còn là if-optional).

### Gap 2 — crystallize Step 10: task files không có context_required cho ui-direction
**File**: `~/.cursor/viepilot/workflows/crystallize.md` Step 10  
**Hiện tại**: Tạo task files theo template, không có rule nhúng ui-direction path.  
**Vấn đề**: vp-auto chỉ đọc `context_required files from task file` — nếu task file thiếu entry ui-direction, execution hoàn toàn không có design context.  
**Fix**: Khi Step 1A tìm thấy ui-direction artifacts, Step 10 phải inject vào mỗi task file UI-related:
```yaml
context_required:
  - .viepilot/ui-direction/{session-id}/notes.md
  - .viepilot/ui-direction/{session-id}/style.css
  - .viepilot/ui-direction/{session-id}/pages/{relevant-slug}.html  # nếu multi-page
```

### Gap 3 — vp-auto Step 3a: không có reminder đọc ui-direction
**File**: `~/.cursor/viepilot/workflows/autonomous.md` Step 3a Load Task Context  
**Hiện tại**: List cố định (AI-GUIDE, TRACKER, PHASE-STATE, task file, context_required).  
**Vấn đề**: Không có safety net — nếu task file thiếu context_required, không có cảnh báo.  
**Fix**: Thêm rule: "Nếu task là UI task (có `ui: true` hoặc task file reference `.viepilot/ui-direction/`), kiểm tra `context_required` có ui-direction — nếu thiếu, warn và tự load `notes.md` từ session gần nhất."

### Gap 4 — AI-GUIDE.md template: thiếu ui-direction entry
**File**: `~/.cursor/viepilot/templates/project/AI-GUIDE.md`  
**Hiện tại**: Quick Lookup table không có row cho ui-direction.  
**Fix**: Thêm row: `| Xem UI layout/design intent | .viepilot/ui-direction/{session}/notes.md | ## Pages inventory |`
