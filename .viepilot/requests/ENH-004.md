## Summary
`vp-docs` khi generate docs cần đọc `git remote get-url origin` và thay thế tất cả placeholder GitHub URLs (`your-org`, `YOUR_USERNAME`, `YOUR_ORG`) bằng URL thực tế của repo.

## Current Feature
`skills/vp-docs/SKILL.md` — Documentation generation skill

## Current Behavior
Khi generate `docs/getting-started.md`, `docs/dev/contributing.md`, và các file khác, `vp-docs` dùng template hardcode:
```
git clone https://github.com/your-org/viepilot.git
git clone https://github.com/YOUR_USERNAME/viepilot
```

Không đọc `git remote` để biết URL thực tế của project.

## Actual vs Expected
| File | Placeholder | Should Be |
|------|-------------|-----------|
| `docs/getting-started.md:23` | `your-org/viepilot` | `0-CODE/viepilot` |
| `docs/dev/contributing.md:10` | `YOUR_USERNAME/viepilot` | `0-CODE/viepilot` |

## Desired Improvement

### Trong `workflows/documentation.md` — đầu workflow, thêm step "Resolve Project Context":
```markdown
### Step 0: Resolve Project Context

Trước khi generate bất kỳ file nào, thu thập context thực tế:

```bash
# Get GitHub repo info
REMOTE_URL=$(git remote get-url origin 2>/dev/null || echo "")
# Extract owner/repo from URL (handles both HTTPS and SSH)
# https://github.com/0-CODE/viepilot.git → 0-CODE/viepilot
# git@github.com:0-CODE/viepilot.git → 0-CODE/viepilot
GITHUB_SLUG=$(echo "$REMOTE_URL" | sed 's|.*github.com[:/]||; s|\.git$||')
GITHUB_OWNER=$(echo "$GITHUB_SLUG" | cut -d'/' -f1)
GITHUB_REPO=$(echo "$GITHUB_SLUG" | cut -d'/' -f2)
```

Dùng `$GITHUB_OWNER/$GITHUB_REPO` thay cho mọi placeholder trong templates.
```

### Trong `skills/vp-docs/SKILL.md` — `<process>`:
Thêm note vào bước generate từng file:
```
> Use resolved GITHUB_OWNER/GITHUB_REPO — never hardcode 'your-org' or 'YOUR_USERNAME'
```

### Post-generation validation:
Sau khi generate xong, scan và warn nếu còn placeholder:
```bash
PLACEHOLDERS=$(grep -r "your-org\|YOUR_USERNAME\|YOUR_ORG\|your-username" docs/ --include="*.md" -l)
if [ -n "$PLACEHOLDERS" ]; then
  echo "⚠️ WARNING: Placeholder URLs found in: $PLACEHOLDERS"
fi
```

## Value Proposition
Docs được generate ra ngay lập tức có URL đúng, không cần manual fix sau.
